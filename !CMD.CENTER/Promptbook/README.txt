HOW TO USE:
-----------
1. Start with 01_RECONCILE (easiest)
2. Progress through 02_IMPORT
3. Advance to 03_EXPORT
4. Test 04-05 for additional validation
5. Use mgmt/ prompts for OPS-level testing

TESTING WORKFLOW:
-----------------
For each prompt:
  1. Open prompt file in OPS.STATION/prompt.book/
  2. Copy PROMPT section
  3. Test in OBSIDIAN.BASE (WITH .3ox)
     → Should use tools
     → Should reference rules
     → Should show LIGHTHOUSE personality
     
  4. Test in SYNTH.BASE (WITHOUT .3ox)
     → Generic AI response
     → No tools
     → No framework features
     
  5. Compare and document differences

PROGRESSIVE DIFFICULTY:
-----------------------
Level 1 [RECONCILE]: Single validation operation
Level 2 [IMPORT]: Multi-file with tagging
Level 3 [EXPORT]: Full workflow with linking + MOC

Each level adds complexity and tests more framework features.
