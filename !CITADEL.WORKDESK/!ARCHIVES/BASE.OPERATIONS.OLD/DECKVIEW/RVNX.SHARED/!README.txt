This is where RVNx (RDP) shares files with other stations.

TO SHARE A FILE FROM RVNx:
  1. Access CMD.BRIDGE (P:\!CMD.BRIDGE)
  2. Navigate to: !BASE.OPERATIONS\DECKVIEW\RVNX.SHARED\
  3. Place file here
  4. Other stations can access via CMD.BRIDGE

TO ACCESS FROM OTHER STATIONS:
  • From OBSIDIAN: Access CMD.BRIDGE → !BASE.OPERATIONS\DECKVIEW\RVNX.SHARED\
  • From SYNTH: Access CMD.BRIDGE → !BASE.OPERATIONS\DECKVIEW\RVNX.SHARED\

Files here are visible to all stations through CMD.BRIDGE.
