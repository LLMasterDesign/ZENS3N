This is where AI (SYNTH) coordination happens.

FOLDERS:
  • REQUESTS\  - CMD.BRIDGE puts requests for AI here
  • RESULTS\   - AI puts completed work here
  • STATUS\    - System status and updates

WORKFLOW:
  1. User puts request in REQUESTS\
  2. AI processes (when opened)
  3. AI puts results in RESULTS\
  4. User retrieves from RESULTS\

All stations can access via CMD.BRIDGE.
