This is where OBSIDIAN (PC) shares files with other stations.

TO SHARE A FILE FROM OBSIDIAN:
  1. Access CMD.BRIDGE (P:\!CMD.BRIDGE)
  2. Navigate to: !BASE.OPERATIONS\DECKVIEW\OBSIDIAN.SHARED\
  3. Place file here
  4. Other stations can access via CMD.BRIDGE

TO ACCESS FROM OTHER STATIONS:
  • From RVNx: Access CMD.BRIDGE → !BASE.OPERATIONS\DECKVIEW\OBSIDIAN.SHARED\
  • From SYNTH: Access CMD.BRIDGE → !BASE.OPERATIONS\DECKVIEW\OBSIDIAN.SHARED\

Files here are visible to all stations through CMD.BRIDGE.
