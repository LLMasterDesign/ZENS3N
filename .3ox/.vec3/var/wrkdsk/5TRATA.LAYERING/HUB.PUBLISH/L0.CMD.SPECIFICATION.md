# L0 CMD Specification

Version: 1.0
Status: Active
Layer: 0 (CMD)
Date: 2026-02-13

## Scope
Layer 0 governs root command layout only. It does not govern file payload content.

## Root Set
- `/root/!ZENS3N.CMD/_TRON`
- `/mnt/v/!CENTRAL.CMD/_TRON`
- `/mnt/r/!CMD.RELAY`

## Required `_TRON` Top-Level Directories
- `Agents`
- `Bases`
- `Services`
- `Stations`
- `systemd`
- `3OX.Ai`

## Relay Integrity
`/mnt/r/!CMD.RELAY` SHALL expose:
- `!CMD.BRIDGE`
- `!CMD.CENTER`
- `!CMD.HUB`
- `_meta`

## Lock Behavior
- Layer 0 lock SHALL enforce required top-level directories.
- Unknown top-level `_TRON` directories SHALL be rerouted, not merged.
- Layer 0 lock SHALL record state and receipts on each run.

## Identification and Meta
The following identifiers are in scope for Layer 0 tracking:
- `CMD.RELAY.ID`
- `CMD.BRIDGE.ID`
- `CMD.CENTER.ID`
- `HUB.ID`

## Non-Goals
- No content deduplication.
- No runtime process tuning.
- No service/agent behavior tuning.
