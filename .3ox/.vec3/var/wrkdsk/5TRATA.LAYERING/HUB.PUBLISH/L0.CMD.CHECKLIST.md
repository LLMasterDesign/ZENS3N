# L0 CMD Checklist

## Pre-Check
- Confirm the three Layer 0 roots are accessible.
- Confirm no other layer lock process is running.

## Validation
1. Run:
```bash
/root/!ZENS3N.CMD/_TRON/systemd/service/5trata/layer-0/layer0_lock.sh --dry-run
```
2. Confirm only top-level layout actions are shown.

## Apply
1. Run:
```bash
/root/!ZENS3N.CMD/_TRON/systemd/service/5trata/layer-0/layer0_lock.sh
```
2. Confirm state:
```bash
cat /root/!ZENS3N.CMD/_TRON/systemd/service/5trata/LAYER.TRACK/layer0/state/layer0.state.json
```
3. Confirm receipts:
```bash
tail -n 20 /root/!ZENS3N.CMD/_TRON/systemd/service/5trata/LAYER.TRACK/layer0/receipts/layer0.receipts.jsonl
```

## Drift Response
- Any unexpected top-level `_TRON` folder SHALL be moved to:
- `/root/!ZENS3N.CMD/!LAUNCHPAD/!1N.3OX ZENS3N/LAYER.REROUTE/layer0`

## Exit Condition
- Layer 0 remains stable over repeated checks with no unresolved reroutes.
