# Layer 1 Base Lock (5TRATA)

Layer 1 controls top-level folder layout for:

- `/root/!ZENS3N.CMD/!LAUNCHPAD`

## Allowed Top-Level Directories

- `!1N.3OX ZENS3N`
- `!CMD.CENTER`
- `!WORKDESK`
- `!ZENS3N.OPS`
- `.3ox`
- `.cursor`
- `.git`
- `.vscode`
- `Antigravity`
- `codex53-work`

Any unexpected top-level directory is moved to:

- `/root/!ZENS3N.CMD/!LAUNCHPAD/!1N.3OX ZENS3N/LAYER.REROUTE/layer1`

## Run

```bash
/root/!ZENS3N.CMD/_TRON/systemd/service/5trata/layer-1/layer1_lock.sh
```

Dry run:

```bash
/root/!ZENS3N.CMD/_TRON/systemd/service/5trata/layer-1/layer1_lock.sh --dry-run
```
