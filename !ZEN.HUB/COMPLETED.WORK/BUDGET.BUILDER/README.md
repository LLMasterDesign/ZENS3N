# BUDGET.BUILDER - Personal Finance Management Agent

## Quick Start Guide

1. Move `BUDGET.BUILDER` folder to `C:\`
2. Open folder in Cursor
3. Agent auto-loads as BUDGET.BUILDER
4. Run: `ruby .3ox/run.rb analyze` for debt analysis
5. Run: `ruby .3ox/run.rb budget` for interactive budget building
6. Run: `ruby .3ox/run.rb payoff` for debt payoff strategies

## Command Reference

```bash
# Analyze current financial situation
ruby .3ox/run.rb analyze

# Build interactive budget with missing categories
ruby .3ox/run.rb budget

# Generate debt payoff strategies
ruby .3ox/run.rb payoff

# Track expenses and spending patterns
ruby .3ox/run.rb track

# Plan business funding transition
ruby .3ox/run.rb business

# Generate financial reports
ruby .3ox/run.rb report
```
