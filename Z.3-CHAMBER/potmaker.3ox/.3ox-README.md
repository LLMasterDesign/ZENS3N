## What is .3ox?

The `.3ox` directory is the standard configuration location for 3OX.AI agents. It contains:

1. **Identity** - Who/what the agent is
2. **Components** - Which subsystems are active
3. **Behavior** - How the agent operates
4. **Constraints** - Limits and boundaries
5. **Schemas** - Process cycles and flows
