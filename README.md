# !ZENS3N.CMD Repository

**Purpose:** VPS deployment repository for Zens3n  
**Git Remote:** `git@github.com:LLMasterDesign/ZENS3N.git`

## Structure

### !ZENS3N.VPS/
VPS deployment structure for Hetzner hosting and live scripts
- `code/` - Git-tracked code for VPS
- `config/` - VPS-specific configs (local only, not in Git)
- `_Runtime/` - Runtime artifacts (logs, pid, cache)
- `scripts/` - Deployment scripts

### !ZEN.CENTER/
Showcase of completed work (capps) - like CMD.CENTER but for completed projects
- `COMPLETED.WORK/` - All completed projects in all caps folders

### ZENS3N.BASE/
Active working space (local, untracked)
- Where active development happens
- Merged from legacy `ZENS3N/` folder
